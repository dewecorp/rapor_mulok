# Rapor Mulok Digitas (RMD)

Aplikasi sistem rapor mata pelajaran muatan lokal untuk MI Sultan Fattah Sukosono.

## Instalasi

1. Import database dari file `database/schema.sql`
2. Konfigurasi database di `config/database.php`
3. Akses aplikasi melalui `http://localhost/rapor-mulok/`

## Login Default

- **Username:** admin
- **Password:** admin123

## Fitur

- Manajemen data madrasah, guru, siswa, dan kelas
- Pengelolaan materi muatan lokal
- Input dan cetak rapor
- Backup dan restore database
- Multi-role: Proktor, Wali Kelas, Guru

## Teknologi

- PHP 7.4+
- MySQL/MariaDB
- Bootstrap 5
- DataTables
- SweetAlert2
- Toastr
